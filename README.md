# REDCapExtractR

The goal of REDCapExtractR is to provide a simple interface to:

- extract records and metadata from a REDCap project
- use the metadata to label the REDCap records

## Installation

You can install the development version of REDCapExtractR like so:

``` r
pak::pak("jatotterdell/REDCapExtractR)
```
